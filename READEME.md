# Store Management System

A Python-based store management system that manages employees, customers, products, and sales using object-oriented programming and SQLite.

## Features

- Employee management
- Customer management
- Product inventory management
- Sales processing
- Inventory updates after sales
- Transaction logging

## Technologies Used

- Python
- SQLite
- Object-Oriented Programming
- Git

## Project Structure

├── models
